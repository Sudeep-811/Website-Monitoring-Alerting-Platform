# Production Ops Platform

A production-grade DevOps platform demonstrating real-world operations,
monitoring, alerting and incident response on AWS.

## Application
**ShopEase API** — A fake e-commerce backend used as the monitored application.

## Stack
- **Infrastructure:** Terraform + AWS (EC2, VPC, IAM, S3)
- **Config Management:** Ansible
- **CI/CD:** Jenkins
- **App:** Python Flask E-Commerce API (Dockerized)
- **Monitoring:** Prometheus + Grafana
- **Alerting:** Alertmanager
- **Logging:** AWS CloudWatch
- **Secrets:** AWS Secrets Manager
