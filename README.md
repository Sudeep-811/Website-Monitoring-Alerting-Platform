# Website Monitoring & Alerting Platform

A production-grade monitoring and alerting platform built on AWS.

## Stack
- **Infrastructure:** Terraform + AWS (EC2, VPC, IAM, S3)
- **Config Management:** Ansible
- **CI/CD:** Jenkins
- **App:** Python Flask (Dockerized)
- **Monitoring:** Prometheus + Grafana
- **Alerting:** Alertmanager
- **Logging:** AWS CloudWatch
- **Secrets:** AWS Secrets Manager

## Project Phases
1. GitHub Repository Structure
2. Terraform AWS Infrastructure
3. Ansible Server Configuration
4. Docker Flask Application
5. Jenkins CI/CD Pipeline
6. Prometheus + Grafana Dashboards
7. Alertmanager + CloudWatch + Secrets Manager
8. Failure Scenarios & Incident Response
9. Documentation & Interview Prep
